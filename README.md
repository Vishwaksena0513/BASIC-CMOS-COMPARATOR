# BASIC-CMOS-COMPARATOR
" Design and simulation of a Basic CMOS Comparator using Cadence Virtuoso software .The project includes schematic design layout design ,and simulation results. "
This project demonstartes the design and simulation of a Basic CMOS Comparator using Cadence Virtuoso.The aim is to compare two analog input voltages and prouduce 
a digit output

## Tools Used 
-Cadence Virtuoso
-CMOS Technology

## Project Steps 
1.Designed CMOS comparator schematic in cadence Virtuoso
2.Performed DC and transient simulations
3.Verified comparator switching behaviour
4.Observed output waveform characteristics

##Results
The comparator successfully compares input voltages and generates a digital output based on the higher input signal
